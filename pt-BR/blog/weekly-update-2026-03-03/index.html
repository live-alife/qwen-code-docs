Qwen Code Semanal: Análise Insight, Imagens da Área de Transferência, Terminal Capture

Qwen Team
2026-03-03

Esta semana lançamos as versões principais v0.10.0, v0.11.0 e 5 versões de correção de bugs.

  • Lançamento de duas versões principais: v0.10.0 e v0.11.0, mais 5 versões de correção de bugs
  • Autenticação OAuth com atualização para qwen3.5-plus, 1000 chamadas gratuitas por dia
  • Adição de análise Insight
  • Suporte a imagens da área de transferência
  • Terminal Capture validação automatizada e geração de capturas de tela

✨ Novos Recursos

Análise Insight

Comando integrado oficial que analisa profundamente os registros de Session locais para fornecer insights de programação personalizados e relatórios de análise de uso.

Uso: Digite /insight para iniciar a análise LLM paralela

Gera um relatório HTML detalhado dividido em 7 seções:

  • What You Work On - Seu foco de trabalho
  • How You Use Qwen Code - Como você usa o Qwen Code
  • Impressive Things - Conquistas notáveis
  • Where Things Go Wrong - Áreas problemáticas
  • Features to Try - Recursos para experimentar
  • New Usage Patterns - Novos padrões de uso
  • On the Horizon - Perspectivas futuras

Finalmente, você pode exportar uma imagem para compartilhamento em redes sociais.

👇 Demonstração do Recurso

Suporte a Imagens da Área de Transferência

Suporte multiplataforma: macOS / Windows / Linux. Veja PR #1612 

Recursos:

  • Captura de Imagem: Capturar imagens da área de transferência do sistema
  • Armazenamento Temporário: Salvar no diretório temporário da área de transferência
  • Limpeza LRU: Excluir automaticamente as 50 imagens mais antigas ao exceder 100
  • UI de Anexos: ↑ para entrar no modo de anexo, ←/→ para navegar, Delete para remover

Terminal-Capture

Propósito: Ferramenta automatizada de captura de tela do terminal para verificação visual CLI, valida automaticamente recursos e produz capturas de tela de demonstração.

Arquitetura: node-pty (pseudo-terminal) → xterm.js (renderização) → Playwright (captura de tela), veja PR #1840
